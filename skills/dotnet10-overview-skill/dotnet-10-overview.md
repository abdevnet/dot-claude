# .NET 10 Release Announcement

**Release Date:** November 11, 2025
**Support Level:** Long Term Support (LTS)
**Support Duration:** 3 years (until November 10, 2028)

## Summary

Microsoft announced .NET 10 as "the most productive, modern, secure, intelligent, and performant release" to date. As a Long Term Support release, organizations can depend on it for extended production support.

## Download

Available at [get.dot.net/10](https://get.dot.net/10)

---

## Key Release Characteristics

### Productivity
- New language features in C# 14 and F# 10
- Simplified property declarations with field-backed properties
- Extension properties for external types

### Modern
- Unified AI integration through Microsoft.Extensions.AI
- Model Context Protocol (MCP) support for standardized tool integration
- Native async/await improvements

### Secure
- Built-in security best practices
- Passkey authentication support in ASP.NET Core
- Enhanced cryptographic capabilities

### Intelligent
- Microsoft Agent Framework for multi-agent systems
- Integrated AI abstractions across the platform

### Performant
- JIT compiler improvements
- Hardware acceleration support (AVX10.2, Arm64 SVE)
- NativeAOT enhancements
- Memory optimization for Arm64

## Coordinated Releases

- **Visual Studio 2026** - Released simultaneously
- **C# Dev Kit** - Updated alongside
